# Dutch Legislation Complexity (DLC)
A public website to make a research accessible for the public. Based on the research '[Paving A Path To Simplify Legislation Using AI](https://github.com/TimvandenBelt/Complexity-Dutch-Legislation)'.

## Goal
The goal of the website is to enable the public to view details of the complexity of the legislation in a clear manner, without needing deep understanding of the academic basis behind it. The website should assist legislators in identifying problematic legislation and view the improvements possible to simplify the legislation.
